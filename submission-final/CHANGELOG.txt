VERSION 2.0.0 - Final Hackathon Submission (2026-02-05)
================================================================================

MAJOR CHANGES:
- Complete AI chatbot intelligence upgrade
- Full natural language task management
- Kubernetes deployment ready
- Production-quality error handling

NEW FEATURES:
- Added "uncomplete task" chatbot command
- Added "clear completed tasks" chatbot command
- Added AI status endpoint (/api/v1/ai/status)
- Added persistent chat history with PostgreSQL
- Added MCP-style tool execution in AI agent
- Added tool action display in chat UI
- Added quick action buttons for common commands
- Added demo mode (works without Cohere API key)
- Added general conversation support via Cohere AI

BUG FIXES:
- Fixed AI router not mounted (missing /api/v1/ai/* endpoints)
- Fixed ChatKitWrapper showing "OpenAI GPT-4o" instead of "Cohere AI"
- Fixed quick action race condition in chat UI
- Fixed missing IntentRouter and AIClient exports from ai module
- Fixed logout endpoint to not require authentication
- Fixed CORS issues for cookie-based authentication
- Fixed SSL parameter compatibility with asyncpg

IMPROVEMENTS:
- Updated app title from "Todo API" to "PakAura API"
- Improved demo mode to support all task operations
- Improved fallback responses with complete command help
- Added uncomplete_task to MCP task operations
- Added clear_completed to MCP task operations
- Improved help command responses with full feature list
- Updated tool display names, icons, and colors in frontend
