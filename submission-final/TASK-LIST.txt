API Endpoints (22 total)
================================================================================

Auth:
  POST   /api/v1/auth/register     - Create new user
  POST   /api/v1/auth/login        - Authenticate user
  POST   /api/v1/auth/logout       - Clear session
  GET    /api/v1/auth/me           - Get current user

Tasks:
  GET    /api/v1/tasks             - List tasks (with ?completed filter)
  POST   /api/v1/tasks             - Create task
  GET    /api/v1/tasks/{id}        - Get single task
  PATCH  /api/v1/tasks/{id}        - Update task
  DELETE /api/v1/tasks/{id}        - Delete task
  POST   /api/v1/tasks/{id}/complete    - Mark complete
  POST   /api/v1/tasks/{id}/uncomplete  - Mark incomplete

AI:
  GET    /api/v1/ai/status         - AI service status
  POST   /api/v1/ai/chat           - Legacy AI chat

Chat:
  POST   /api/{user_id}/chat              - AI chat with history
  GET    /api/{user_id}/chat/history      - Get chat history
  DELETE /api/{user_id}/chat/history      - Clear chat history

System:
  GET    /api/v1/health            - Health check
  GET    /                         - Root endpoint
  GET    /api/docs                 - Swagger UI
  GET    /api/redoc                - ReDoc
